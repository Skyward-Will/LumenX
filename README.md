# LumenX - 基于 Deepin 的 Linux 发行版

[![GitHub License](https://img.shields.io/github/license/Skyward-Will/LumenX)](https://github.com/Skyward-Will/LumenX/blob/main/LICENSE)
[![Deepin 25.0.1 Base](https://img.shields.io/badge/Base-Deepin%2025.0.1-blue)](https://www.deepin.org/index/en)

LumenX 是一个基于深度操作系统 (Deepin) 的 Linux 发行版，旨在为国内用户提供一个更轻量、更易定制且专注于用户体验的桌面环境。本项目目前处于早期开发阶段。

**项目仓库:** https://github.com/Skyward-Will/LumenX

## ✨ 主要特性

*   **本土化优化:** 默认适配国内用户的使用习惯，预装了常用的软件和字体。
*   **轻量简洁:** 在 Deepin 的基础上进行了一些精简，力求系统运行更流畅。
*   **易于定制:** 修改了系统标识和主题，提供了独特的 LumenX 视觉体验。
*   **开箱即用:** 集成了必要的驱动和解码器，满足日常办公、娱乐和开发需求。
*   **安全透明:** 遵循开源理念，采用 GPLv3 许可证，代码公开可查。

## 📥 下载与安装

**请注意：LumenX 目前仍在开发中，尚未提供正式的安装镜像。**

敬请关注本项目的 [GitHub Releases 页面](https://github.com/Skyward-Will/LumenX/releases)以获取未来的版本更新和下载信息。

未来的安装步骤大致如下：
1.  从 Releases 页面下载最新的 LumenX ISO 镜像文件。
2.  使用 Etcher、Rufus 等工具将镜像写入 U 盘。
3.  从 U 盘启动计算机，进入 Live 环境。
4.  运行安装程序，按照提示完成系统安装。
